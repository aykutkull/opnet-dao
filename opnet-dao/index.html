About OPNet DAO

OPNet DAO is a fully on-chain governance protocol running on Bitcoin Layer 1 via OPNet. GOV token holders propose, vote on, and execute changes directly on Bitcoin — no sidechains, no bridges, no compromises.

  • Proposals are open for voting for 144 blocks (~1 day on Bitcoin)
  • A proposal passes when For votes exceed Against votes at voting end
  • Passed proposals can be executed by any member after they succeed
  • Proposers can cancel their own proposals before execution
  • Smart contracts compiled to WebAssembly, deployed directly on Bitcoin L1
  • Governance power is proportional to GOV token holdings; delegate freely
